zimbra-perl-soap
================
Zimbra administration with perl and SOAP.

Use Case
--------

Installation
------------

Build infrastructure to run the framework

    $ cd /opt
    $ git clone https://github.com/rplessl/zimbra-perl-soap
    $ cd zimbra-perl-soap
    $ ./setup/build-perl-modules.sh

Usage
-----
    $ ./bin/zimbra-admin-interface.pl
